# Flappy Bird Game Project

## Overview
Flappy Bird is a side-scrolling mobile game developed by Dong Nguyen. The player's goal is to direct a flying bird, which is in constant free fall, between columns of green pipes. The game is known for its challenging gameplay and simplistic mechanics.

## Features
- Simple and intuitive controls
- Challenging gameplay minimizing obstacles
- High-score tracking
- Beautiful graphics and sound effects

## Installation
To clone the project, run:

```bash
git clone https://github.com/Tsaglashgui-pro/Flappy-Bird.git
```

## How to Play
1. Tap the screen to make the bird jump.
2. Avoid hitting the pipes.
3. Try to achieve the highest score!
